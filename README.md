# GestiAct: Sistema de Gestión de Activos

GestiAct es una solución web integral diseñada para la administración, control y seguimiento de activos empresariales. El sistema permite gestionar activos, categorías y usuarios de manera eficiente, ofreciendo una interfaz moderna y un dashboard intuitivo.

## Características principales
- **Dashboard en tiempo real:** Visualización gráfica de la distribución de activos por categoría.
- **Gestión CRUD completa:** Creación, lectura, actualización y eliminación de activos.
- **Seguridad:** Protección de rutas mediante AuthGuard.
- **Diseño Responsivo:** Interfaz construida con Tailwind CSS.

## Tecnologías Utilizadas
- **Frontend:** Next.js, React, Tailwind CSS, Chart.js.
- **Backend:** Node.js, Express.
- **Base de Datos:** Supabase (PostgreSQL).
- **Despliegue:** Netlify (Frontend) / Render (Backend).
